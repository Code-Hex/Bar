    go get -u github.com/Code-Hex/battery/cmd/battery

## Usage
For tmux user, please write `#(battery -t)` in your `.tmux.conf`  
Please refer to [this](https://github.com/Code-Hex/dotfiles/blob/master/tmux/.tmux.conf#L82)
